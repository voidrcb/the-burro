# Part 2: Getting Started - Operator Login

## How to Log In

Follow these steps exactly:

### Step 1: Open Your Web Browser

Your web browser is the app you use to visit websites. Common ones are:
- **Chrome** (has a colorful circle icon)
- **Edge** (blue "e" icon on Windows)
- **Safari** (compass icon on Mac/iPhone)

Click on whichever browser you normally use.

### Step 2: Go to the Login Page

At the top of your browser, there is a long white box called the address bar.

1. Click inside the address bar
2. Delete anything already there
3. Type exactly: `bigbendburro.vcetexas.com/login`
4. Press the Enter key on your keyboard

The login page will appear.

### Step 3: Enter Your Username

On the login page, you will see a box that says "Username" or "Email."

1. Click inside that box
2. Type your username:
   - Chuck: type `chuck`
   - Susan: type `susan`
   - If using the shared account: type `operator`

### Step 4: Enter Your Password

Below the username box, there is a password box.

1. Click inside the password box
2. Type your password (Ryan gave this to you)
3. Note: The password will show as dots for security

**Important:** Passwords are case-sensitive. If your password is "BigBend123" then "bigbend123" will NOT work.

### Step 5: Click Sign In

Below the password box, there is a button that says "Sign In."

1. Click the "Sign In" button
2. Wait a moment for the page to load
3. You should see the operator dashboard

**You are now logged in.**

When you are done, always log out. This keeps your account secure.

## Equipment Inspection Workflow

Always inspect equipment before and after every rental. This protects you and the customer.

### Pre-Rental Checklist

Before handing over equipment:

1. **Check hour meter reading** - Write down the number
2. **Inspect for visible damage** - Walk around, look at everything
3. **Test all controls** - Make sure everything works
4. **Check fluid levels** - Oil, hydraulic fluid, fuel
5. **Take photos** - Front, back, both sides, any existing damage
6. **Record condition in system** - Note everything

### Post-Rental Checklist

When equipment returns:

1. **Compare hour meter** - Calculate hours used
2. **Inspect for new damage** - Compare to pre-rental photos
3. **Check fluid levels** - Note any unusual consumption
4. **Document any issues** - Write down what you find
5. **Take return photos** - Same angles as before
6. **Calculate charges** - Rental time + any damage fees

### Using the Mobile Inspection Tool

There is an inspection form at `/assistant/rentals/inspect`:

1. Open the inspection page
2. Select the rental/equipment
3. Fill out the checklist
4. Take and upload photos
5. Record readings
6. Submit the report

### Equipment Maintenance

Keep equipment in good condition:

1. **View maintenance notes** - See what's been done
2. **Record maintenance performed** - Oil changes, repairs, etc.
3. **Schedule service reminders** - Don't miss maintenance intervals

| **Logout** | Signing out when you're done |
| **Refresh** | Reloading a page to see the latest information |
| **Cache** | Saved copies of pages that can sometimes show old information |
| **Session** | A specific date/time when a workshop happens |
| **Capacity** | How many people can attend a session |
| **Waiver** | Legal form guests accept before activities |
| **CMS** | Where website content is stored (you don't need to touch this) |
| **API** | How different systems talk to each other (technical - Ryan handles this) |
| **Lodgify** | The booking service that handles stay payments |
| **Stripe** | The payment service that will handle shop and workshop payments |
