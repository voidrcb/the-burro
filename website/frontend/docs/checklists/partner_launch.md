# Partner Launch Checklist

**Purpose:** Verify readiness before activating a new partner on the platform.

---

## Pre-Launch (Partner Onboarding)

- [ ] **(blocking)** Partner agreement signed - Operator
  - Verify: Signed contract in partner file

- [ ] **(blocking)** Legal name and contact info verified - Operator
  - Verify: PartnerRecord has complete contact fields

- [ ] **(blocking)** Insurance documentation received (if required) - Operator
  - Verify: liability.insuranceVerified = true

- [ ] **(advisory)** Partner orientation call completed - Operator
  - Verify: Notes added to partner record

---

## Content Preparation

- [ ] **(blocking)** Partner bio/description approved - Operator
  - Verify: Content reviewed for accuracy and brand alignment

- [ ] **(blocking)** Experience listings created (if applicable) - Operator
  - Verify: ExperienceProduct records linked to partner

- [ ] **(blocking)** Photos meet quality standards - Operator
  - Verify: Images are high-res, properly cropped

- [ ] **(advisory)** Partner preview approved by partner contact - Partner
  - Verify: Email confirmation received

---

## System Configuration

- [ ] **(blocking)** Commission rules configured - Operator
  - Verify: CommissionRule exists for partner

- [ ] **(blocking)** Partner tier assigned - Operator
  - Verify: governance.tier set appropriately

- [ ] **(blocking)** Marketplace items in approved state - Operator
  - Verify: MarketplaceItem.publicationState = 'approved'

- [ ] **(advisory)** Communication preferences confirmed - Partner
  - Verify: contact.preferredLanguage set

---

## Go-Live

- [ ] **(blocking)** Partner status set to approved - Operator
  - Verify: approvalStatus = 'approved'

- [ ] **(blocking)** Marketplace items published - Operator
  - Verify: publicationState = 'published'

- [ ] **(advisory)** Welcome email sent - System
  - Verify: Postmark delivery confirmed

- [ ] **(advisory)** Social media announcement scheduled - Operator
  - Verify: Content calendar updated

---

## Post-Launch (Week 1)

- [ ] **(advisory)** Partner dashboard access confirmed - Partner
  - Verify: Partner can view their content

- [ ] **(advisory)** First booking/transaction monitored - Operator
  - Verify: Analytics show activity

- [ ] **(advisory)** Partner feedback collected - Operator
  - Verify: Check-in call scheduled
