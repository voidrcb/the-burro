# Content Publish Checklist

**Purpose:** Verify quality before publishing major content updates.

---

## Content Quality

- [ ] **(blocking)** Spelling and grammar checked - Author
  - Verify: No errors in title, body, captions

- [ ] **(blocking)** Factual accuracy verified - Operator
  - Verify: Dates, names, prices correct

- [ ] **(blocking)** Links tested - Author
  - Verify: All internal/external links work

- [ ] **(advisory)** Tone matches brand voice - Operator
  - Verify: Consistent with existing content

---

## Visual Assets

- [ ] **(blocking)** Images properly sized - Author
  - Verify: No oversized or pixelated images

- [ ] **(blocking)** Alt text provided - Author
  - Verify: Accessibility attributes present

- [ ] **(advisory)** Image credits added if required - Author
  - Verify: Attribution present for licensed images

---

## Technical Review

- [ ] **(blocking)** Page loads without errors - Operator
  - Verify: No console errors, proper rendering

- [ ] **(blocking)** Mobile display tested - Operator
  - Verify: Content readable on phone

- [ ] **(advisory)** SEO metadata present - Author
  - Verify: Title, description, OG tags set

---

## Bilingual Content (if applicable)

- [ ] **(blocking)** Spanish translation reviewed - Translator
  - Verify: TranslationRecord status = 'approved'

- [ ] **(advisory)** Cultural context appropriate - Translator
  - Verify: No awkward translations

---

## Partner Content (if applicable)

- [ ] **(blocking)** Partner approved content - Partner
  - Verify: Written confirmation received

- [ ] **(blocking)** Contact info accurate - Operator
  - Verify: Partner details current

---

## Pre-Publish

- [ ] **(blocking)** Preview reviewed by second person - Operator
  - Verify: Peer review completed

- [ ] **(advisory)** Publication time optimal - Operator
  - Verify: Scheduled for appropriate time

---

## Publish

- [ ] **(blocking)** Content published - Operator
  - Verify: Live URL accessible

- [ ] **(advisory)** Social media announced - Operator
  - Verify: Posts scheduled/published

- [ ] **(advisory)** Newsletter inclusion considered - Operator
  - Verify: Added to next newsletter if appropriate

---

## Post-Publish

- [ ] **(advisory)** Analytics tracking verified - Operator
  - Verify: Page views registering

- [ ] **(advisory)** Feedback monitored - Operator
  - Verify: Check for comments/questions
