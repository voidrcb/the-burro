# pages

Frontmatter must satisfy the corresponding schema in `website/frontend/lib/content/types.ts`.
- `pages/` -> `Page` view model
- `blog/` -> `BlogPost` view model
- `activism/` -> `ActivismUpdate` mirror contract
- `experiences/` -> `Experience` mirror contract
- `workshops/` -> `Workshop` mirror contract
- `lodging/` -> `LodgingUnit` mirror contract
